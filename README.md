# trivia-scoreboard-backend
Backend app scoreboard for the Great Midwest Trivia Contest

# Setup

Init virtual environment. This localizes python version and package management to root directory.

```shell
# Create virtualenv
python -m venv venv
# Activate
source ./venv/bin/activate
# Test if uncertain
which python # --> should output local venv directory
pip freeze # --> should show project's requirements.txt
```
